# Timeseries_forecasting

## Dataset
- Daily Website Visitors
- Original data : 2100 entries
- Reduced data : 500 entries
- 4 variables (Page Loads, Unique Visits, First Time visits, Returning visits ) 

## Response variable 
- Page Loads
